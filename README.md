# pvp-mixin-base
A PvP Minecraft Client base using Mixins.

# Features
* Module System (normal and draggable).
* Reflect Event System with events (Render2DEvent, TickEvent).
* Keybinding System (weird idea but yeah).
  Config System with Mod config.
* Mixin tweaker.
* Drag hud.
* Modlist Gui.
* Value System
